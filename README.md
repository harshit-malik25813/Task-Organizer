# Task Organizer (C Program)

A simple **command-line Task Organizer written in C** that allows users to manage tasks directly from the terminal.

⚠️ **Project Status**
This project is currently **under development and may contain bugs.**  
It is mainly a **learning project** to practice C programming and CLI application design.

---

# Features

- Add tasks
- View tasks
- Update tasks
- Delete tasks
- Simple menu-driven CLI interface

---

# Technologies Used

Language:
- C

Concepts Practiced:
- Structures
- Functions
- File handling
- Loops and conditional logic
- Menu-driven CLI programs

# Installation

## Clone the Repository

```bash
git clone https://github.com/harshit-malik25813/Task-Organizer.git
cd Task-Organizer
```

---

# Compilation

## Linux / macOS

```bash
gcc taskcheck.c -o taskorganizer
```

## Windows (MinGW)

```bash
gcc taskcheck.c -o taskorganizer.exe
```

---

# Running the Program

## Linux / macOS

```bash
./taskorganizer
```

## Windows

```bash
taskorganizer.exe
```
